Art Institute of Chicago Artworks Reference

With this application, you can explore the Art Institute of Chicago's collection of artworks, look for specific pieces, and review information about each work. By utilizing the Art Institute of Chicago's API, this application extracts data regarding the artworks, which you can obtain through the API endpoints.

API Integration

If you want to access information about the Art Institute of Chicago's artworks, the API provides a straightforward approach. By sending an HTTP request to the API endpoint(s) and parsing the response data, you can retrieve the information you need. The API documentation offers instructions on how to create requests and extract data.

Here are the API endpoints that the application uses:

  • GET /api/v1/artworks - Retrieves a list of all artworks in the catalog.
  • GET /api/v1/artworks/:id - Retrieves data for a specific artwork based on its ID.
  • GET /api/v1/artworks/search?q=:query - Searches for artworks based on a search query.

You can use these endpoints to fetch data about the artworks and display it in the application.

Using the Application

To use the application, follow these steps:

  1. Open the application in your web browser.
  2. Browse through the list of artworks or use the search feature to find specific artworks.
  3. You can manage your favourites and add, delete any of them.
  4. Click on an artwork to view its details, such as its title, artist, image, and description.

Conclusion

This application provides a convenient way to browse through the Art Institute of Chicago's artworks catalog and view details about each artwork. The application uses the Art Institute of Chicago's API to fetch data about the artworks, which you can access using the API endpoints.
